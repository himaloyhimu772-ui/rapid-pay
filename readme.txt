Manual payment gateway for WooCommerce supporting bKash, Nagad, Rocket, and Upay with comprehensive admin dashboard and analytics.

== Description ==

Rapid Pay is a powerful and user-friendly payment gateway plugin for WooCommerce that enables manual payment processing through popular mobile financial services in Bangladesh including bKash, Nagad, Rocket, and Upay.

= Key Features =

* **Multiple Payment Methods**: Support for bKash, Nagad, Rocket, and Upay
* **Checkout Integration**: Seamlessly integrates with WooCommerce checkout
* **Required Fields**: Collects sender mobile number and transaction ID
* **Order Management**: Dedicated admin dashboard to manage all Rapid Pay orders
* **Status Management**: Easily mark orders as Completed, Refunded, Cancelled, or Pending
* **Analytics Dashboard**: View daily, weekly, monthly, and total earnings
* **Income Chart**: Visual representation of your earnings over time
* **Date Filtering**: Filter orders by date range and status
* **Customizable Settings**: Enable/disable payment methods, set min/max amounts
* **Auto-Expire**: Automatically cancel pending orders after specified hours
* **Custom Instructions**: Add custom payment instructions on checkout
* **Admin Phone Numbers**: Display your payment receiving numbers to customers
* **Translation Ready**: Fully translatable with .pot file included
* **Secure**: Follows WordPress coding standards with proper sanitization and validation
* **Responsive Design**: Works perfectly on all devices

= How It Works =

1. Customer selects Rapid Pay at checkout
2. Customer chooses payment method (bKash, Nagad, Rocket, or Upay)
3. Customer completes payment using their mobile wallet
4. Customer enters sender mobile number and transaction ID
5. Order is placed with "On Hold" status
6. Admin reviews payment details in the dashboard
7. Admin marks order as Completed, Refunded, or Cancelled

= Admin Dashboard Features =

* View all Rapid Pay orders in one place
* See order ID, customer name, phone, transaction ID, amount, method, date, and status
* Quick status updates directly from the dashboard
* Real-time analytics with earnings cards
* Beautiful income chart showing 30-day trend
* Filter orders by status and date range
* Export-ready data table

= Settings Options =

* Enable or disable specific payment methods
* Customize checkout instruction text
* Set admin phone numbers for payment
* Set minimum payment amount
* Set maximum payment amount
* Enable auto-expire for pending orders
* Configure auto-expire duration (1-168 hours)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rapid-pay` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Make sure WooCommerce is installed and activated.
4. Go to WooCommerce > Settings > Payments and enable "Rapid Pay".
5. Go to Rapid Pay > Settings to configure your payment methods and options.
6. Add your payment receiving phone numbers in the settings.
7. Start accepting payments!

== Frequently Asked Questions ==

= Does this plugin automatically verify payments? =

No, this is a manual payment gateway. You need to verify the transaction ID and sender phone number manually before marking orders as completed.

= Can I disable specific payment methods? =

Yes, you can enable or disable bKash, Nagad, Rocket, and Upay individually from the settings page.

= Will orders automatically expire? =

Only if you enable the auto-expire feature in settings. You can set the duration from 1 to 168 hours.

= Can I customize the checkout instructions? =

Yes, you can add custom instructions that will be displayed to customers during checkout.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready with a .pot file included in the languages folder.

= Does it work with the latest WooCommerce version? =

Yes, the plugin is tested and compatible with WooCommerce 8.5 and above.
